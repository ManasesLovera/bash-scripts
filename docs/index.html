bash-scripts

Portable shell scripts for common development tasks.

Quick Install

Method 1: Direct (fastest)

curl -sSL https://raw.githubusercontent.com/ManasesLovera/bash-scripts/main/init-todo.sh | bash

Runs immediately without reviewing the code.

Method 2: Download first (recommended for security)

curl -sSL https://raw.githubusercontent.com/ManasesLovera/bash-scripts/main/init-todo.sh -o init-todo.sh
chmod +x init-todo.sh
./init-todo.sh

Review the script before executing it.

Note: The alias is installed to ~/.bashrc only if you run with --install-global: 
curl -sSL https://raw.githubusercontent.com/ManasesLovera/bash-scripts/main/init-todo.sh | bash -s -- --install-global

Scripts

init-todo.sh

Multi-project todo CLI with SQLite backend. Each project gets its own .todo.db database.

Commands

CommandDescription
todo add "title" [--pri 1-3]Add a task
todo list [--all|--done]List tasks (default: pending only)
todo get <uuid>Get task by ID
todo start <uuid>Mark task in-progress
todo done <uuid>Mark task done
todo update <uuid> --title "..." --desc "..." --pri NUpdate task
todo delete <uuid>Delete task
todo projects listFind all projects
todo projects switch /pathSwitch project
todo projects currentShow current project

Requirements

  • bash 4+
  • sqlite3

Database

Each project stores data in <project_root>/.todo.db.

  • Commit it to share tasks with your team
  • Or add to .gitignore for local-only data
